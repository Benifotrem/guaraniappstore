# Carpeta de Imágenes

Esta carpeta contiene los assets gráficos del sitio.

## Logo Principal

**Ubicación requerida:** `logo.png`

El logo se usa en:
- Landing page (header público)
- Panel admin (sidebar)
- Página de login
- Página de verificación 2FA

**Especificaciones recomendadas:**
- Formato: PNG con transparencia
- Dimensiones: 200x200px (mínimo)
- Fondo: Transparente
- Colores: Tema Guarani (verde Paraguay)

## Otros Assets

- `favicon.png` - Icono del navegador (32x32px)
- `og-image.png` - Imagen para redes sociales (1200x630px)

## Instrucciones

1. Coloca tu logo como `logo.png` en esta carpeta
2. Opcionalmente, añade `favicon.png` y `og-image.png`
3. Los archivos se cargarán automáticamente desde el código PHP
